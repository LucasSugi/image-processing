# Filtragem unidimensional de uma imagem

Para usar o programa deve-se entrar com:
-- nome da imagem
-tipo de filtro(1 - arbitrário, 2- função Gaussiana)
-tamanho do filtro
-pesos do filtro
-dominio da imagem(1 - espacial, 2 - frequencia)

Exemplo de entrada:

cat.png
1
5
0.4 0.2 0.1 0 0.3
1

cat.png
2
7
0.7
2
